# 우리 집 택시

어르신이 큰 버튼으로 자주 가는 목적지를 고르면, **출발지와 목적지를 복사한 뒤 카카오 T를 여는** Android 홈 화면 설치형 웹앱(PWA)입니다.

## 현재 동작

- 병원·시장 등 자주 가는 곳을 큰 버튼으로 표시
- 출발지·목적지를 한 번에 클립보드로 복사
- Android에서 설치된 카카오 T 앱 열기 시도
- 카카오 T 내에서 주소 확인 및 **최종 호출은 직접 실행**
- 가족 도움 요청 문구를 복사/공유하고, 가족 번호가 있으면 SMS 작성 화면 열기
- 주소·가족 연락처는 휴대폰 브라우저의 로컬 저장소에만 보관

> 카카오 T의 개인 택시 호출용 공개 API가 확인되지 않아, 이 앱은 계정/결제/배차를 자동 조작하지 않습니다.

## 설정과 사용

1. 스마트폰 Chrome에서 배포된 주소를 연다.
2. 메뉴에서 **홈 화면에 추가**를 누른다.
3. `주소와 목적지 설정`에서 집 앞 승차 위치와 자주 가는 목적지를 저장한다.
4. 홈 화면의 `우리 집 택시` 아이콘을 눌러 사용한다.

## 실제 배포 전 필요 항목

- 어머님 집 앞의 정확한 승차 위치 이름·주소
- 자주 가는 목적지 2~3곳의 이름·주소
- 도움 요청을 받을 가족 연락처
- Android 휴대폰에서 열 수 있는 HTTPS 주소(예: 개인 웹호스팅)
